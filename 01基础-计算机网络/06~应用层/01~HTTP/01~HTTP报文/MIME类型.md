
## MIME类型
| 数据格式               | 描述                                         |
| ---------------------- | -------------------------------------------- |
| text/plain             | 纯文本，告诉浏览器按照文本格式解释           |
| text/html              | 文本与网页，告诉浏览器按照 html 格式解释内容 |
| text/css               | 样式， 告诉浏览器以 CSS 方式解释             |
| application/javascript | js 代码片断， 告诉浏览器按照 JS 方式运行解释 |
| application/xml        | 按 XML 的方式解释                            |
| application/json       | 按 json 方式解释                             |
| images/jpeg            | 图片                                         |
