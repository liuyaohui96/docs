# Float

## float classes
* `float-left`, `float-right`, `float-none`, and the reposive float
```SCSS
// resposive float: float-((sm)|(md)|(lg)|(xl))-((left)|(right))|(none))
@mixin float-left {
  float: left !important;
}
@mixin float-right {
  float: right !important;
}
@mixin float-none {
  float: none !important;
}

```
