
## 位运算符
```js
a & b // 每一个对应的位都为1则返回1， 否则返回0
a | b // 每一个对应的位，只要有一个为1则返回1， 否则返回0
a ^ b // 每一个对应的位，两个不相同则返回1，相同则返回0
~ a // 反转被操作数的位

a << b // 二进制串向左移动b位,右边移入0
a >> b // 进制表示向右移动b位，丢弃被移出的所有位

// 对非负数值，补零右移和带符号右移产生相同结果。
a >>> b // 二进制表示向右移动b位，丢弃被移出的所有位，并把左边空出的位都填充为0

// === 例子
9<<2: 即1001移位2比特向左变为100100，结果为36
9>>2: 即1001右移2比特向右变为10，即2
-9>>2： 由于符号被保留，结果为-3
```
