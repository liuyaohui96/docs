## 单个或者多个字符的匹配
```js
//=== ? 表示匹配1个或者多个前一个字符 的字符串
liu? // 匹配零个或者1个u的字符串

// === * 表示匹配0个或者多个前一个字符 的字符串
liu* // 匹配0个或者多个u的字符串

// === + 表示匹配一个或者多个前一个字符 的字符串
liu+ // 匹配1个或者多个u的字符串

// {m} 表示匹配m个前一个字符 的字符串
// {m, } 表示至少匹配m个前一个字符 的字符串
// {,n} 表示至多匹配n个前一个字符 的字符串
// {m, n} 表示匹配m个到n个前一个字符 的字符串
liu{m} // 匹配m个u的字符串
liu{m,} // 匹配至少m个u的字符串
liu{m, n} // 匹配m个到n个u的字符串
```
