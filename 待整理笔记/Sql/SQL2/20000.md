## Insert row into table
```SQL
INSERT INTO <Table> VALUES(...);

1. Description: insert one row in table

2. Parameters
     <Table>: [A-Z][a-z0-9_]*
```

## Rename table
```SQL
-- PostgreSQL
ALTER TABLE <Table> RENAME TO <Table>;
1. Description: rename table name

2. Parameters
    <Table>: [A-Z][a-z0-9_]*

-- MySQL
RENAME TABLE <Table> to <table>
```
