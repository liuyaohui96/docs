# Database & DBMS & RDBMS


## Database
Database: store data

## DBMS
DBMS: system of managing database

* HDB(Hierarchical Database)
    
    通过层次结构（树形结构）的方式表现出来,曾是数据库的主流

* RDB(Relational Database)
    
    由行和列组成的二维表来管理数据


* OODB(Object Oriented Database)

    把数据以及对数据的操作集合起来以对象为单位进行管理

* XMLDB(XML Database)

    对XML 形式的大量数据进行高速处理

* KVS(Key-Value Store)

    单纯用来保存查询所使用的主键（Key）和值（Value）的组合的数据库

why DBMS so important?
* Sharing data
* Operatting various data format
* Enable operated by program
* Safe & Maintainable
