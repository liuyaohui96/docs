# SQL data types
```SQL
<sql-data-type>: <number>|<string>|<date>

<number>: <integer>|<floating-number>
    <integer>: BIT|TINYINT|SMALLINT|INT|BIGINT
               |DECIMAL|NUMBERIC|SMALLMONEY|MONEY

    <floating-number>: FLOAT|REAL

<string>: CHAR|CHAR(\d)|VARCHR|VARCHAR(\d)|TEXT
          |NCHAR|NCHAR(\d)|NVARCHAR|NTEXT 

<date>: DATE|TIME|SMALLDATETIME|DATETIME
```
