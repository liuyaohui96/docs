```SQL
<sql-operator>: <arthemetic-operator>|<comparison-operator>
                |<logical-operator>

 <arthemetic-operator>: \+|-|\*|/

 <comparison-operator>:  =|(!=)|(<>)
                           |<|>
                           |(<=)|(!>)
                           |(>=)|(!<)

<legical-operator>: (IS NULL)|UNIQUE
                      |NOT|AND|OR|IN|BETWEEN
                      |LIKE|EXISTS|
```
