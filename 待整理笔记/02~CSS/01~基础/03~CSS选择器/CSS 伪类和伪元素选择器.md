

## CSS 伪类和伪元素选择器
区别：
* 语法：伪类为单冒号`:`，伪元素为`::`
* 用途：
    * 伪类用于定义元素的状态，当一个元素达到一个特定状态时，就会应用指定规则，当状态改变时，它又会失去规则
    * 伪元素用于定义元素的特定内内容，如
        * 元素内容中的第一个字母，第一行
        * 元素的前后内容，用于设置特定内容（如三角形）和添加不占空间的元素

> 起初，伪元素的前缀使用的是单冒号语法，如CSS2.1里`:before 和 :after` 这两个伪元素，但在CSS3的规范里，伪元素的语法被修改成使用双冒号语法，成为`::before 和 ::after`
