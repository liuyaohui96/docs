## 继承
在一个属性应用于一个特定标签之后，该标签的子标签也应用该属性。产生的条件
  * 该属性是可继承的
  * 父级元素设置属性，该子元素没有覆盖该属性
  
> 并不是所有的属性都会被继承。如文字样式一般都继承，盒图的宽高一般不继承。
