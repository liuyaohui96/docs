
## 字符间距和单词间距属性
```css
/* === letter-spacing 指示字符间距 */
letter-spacing: normal;
letter-spacing: 2px; 
letter-spacing: 0.1em; /* 相对于元素字体大小 */

/* === word-spacing 指示单词间距 */
word-spacing: normal;
word-spacing: 5px;
word-spacing: 2em; /* 相对于元素字体大小 */
```

## 单词间距与字母间距属性
word-spacing : 单词

letter-spacing : 字母、中文
