
## CSS 布局
CSS 布局就是控制页面中的元素，使得它们相对于常规布局default position in normal layout flow, 或者相对于周围其他元素，或者父容器，或者主窗口/主适口的相对定位

CSS 布局主要涉及到以下概念：
1. Normal flow
2. Display
3. Flexbox
4. Gridg
5. Floats
6. Positioning
7. Multiple-column layout
8. Responsive design

> Table layout（不建议)

现代公司的开发通常分为：
1. 分离式开发。通常分为移动和pc这两套开发版版本，通常大公司会使用
2. 响应式开发。中大型和个人blog会使用
