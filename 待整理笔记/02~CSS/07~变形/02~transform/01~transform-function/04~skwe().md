## skew()
skew() 函数定义了一个元素在二维平面上的倾斜转换

在水平和垂直方向上将单元内的每个点扭曲一定的角度。一个点离原点越远，其增加的值就越大
```css
/* ax 沿横坐标扭曲元素的角度 */
/* ay 沿纵坐标扭曲元素的角度。如果未定义，则其默认值为0，导致纯水平倾斜。 */
/* 正值向左向上倾斜 */
skew(ax)
skew(ax, ay)
```

## skewX() & skewY()
skewX() & skewY() 在各自方向上进行倾斜
```css
skewX(ax)
skewY(ay)
```
